Models:
    - This component is used to define the database schema or structure in the database.
    - In models.py file we define a class which is the child class of model class.
    - This class will be converted in to a table in the database.
    - class represents table and object represents rows of the table.
    

steps:
1. Create an app
2. Register your app in Settings.py
3. In app in models.py create your model like :

	from django.db import models

	# Create your models here.

	class StudentModel(models.Model):
    		name = models.CharField(max_length = 50)
   			roll = models.IntegerField()
    		subject = models.CharField(max_length = 50)
    		email = models.EmailField()
    		address = models.TextField()
	

4. In terminal run ---> Python manage.py makemigrations
5. ---> Python manage.py migrate
6. Register your model in admin.py in your app like:

	from django.contrib import admin
	from .models import StudentModel

	# Register your models here.
	class StudentModelAdmin(admin.ModelAdmin):
    	list_display = ['name','roll','subject','email','address']
	admin.site.register(StudentModel,StudentModelAdmin) 

7. ---> Python manage.py createsuperuser
