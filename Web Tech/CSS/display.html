Display Properties

  1. inline

  4. flex
    1. It is 1 dimension layout system.

    2. Flex container:

      Parent element with display:flex that controls layout of its children

    3. Flex item:

      Direct children of flex container that get positioned by flex rule.

    4. Flex Axis:

      Primary axis along which flex items are arranged(horizontal by default)

    5. Cross Axis:

      Perpendicular axis to main axis(vertical by default)
    Flex Container property / Flex parent property
    1. display:Flex
    2. Flex-direction:
      • row
      • column
      • row-reverse
      • column-reverse
    3. flex-wrap:
      • nowrap
      • wrap
      • wrap-reverse
    4. flex-flow:flex-direction flex-wrap
    5. gap:row-gap column-gap
    6. justify-content:
      1. flex-start
      2. flex-end
      3. center
      4. space-between
      5. space-around
      6. space-evenly
    7. align-items:
      • stretch
      • flex-start
      • flex-end
      • center
    8. align-content:
      • stretch
      • flex-start
      • flex-end
      • center
      • space-between
      • space-around
      • space-evenly
    Flex Item Property
    1. flex-basis: 100px (value in px, %, rem)
    2. order: 1 (value in number)
    3. align-self:
      • stretch

      • flex-end
      • center
    4. flex-grow: 1 (Value in number)
    5. flex-shrink: 1 (value in number)
  5. grid
  6. none

Differences:

  1. Display: None;

    Element removed from flow,takes no space, not accessible

  2. Visibility: Hidden;

    Hidden but holds space,children can override with visibility: Visible, not accessible

  3. opacity: 0;

    Fully transparent,still takes place, still interactive and supports transition
