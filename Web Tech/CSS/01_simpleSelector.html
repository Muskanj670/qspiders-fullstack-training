Types of Simple Selectors
  • ID: used to target element uniquely, represented as '#'.
  • Class: Used to target multiple element or different element, represented as '.'.
  • Element: Used to target multiple same type of element, represented as tagName.
  • Group: Used to Group multiple elements, represented as ','.
  • Universal: Used to target all elements in HTML, represented as '*'.
