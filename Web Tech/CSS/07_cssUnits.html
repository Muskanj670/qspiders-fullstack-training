Types of units in CSS

  1. px -- Fixed/ Absolute units (1 inch divided by 96)
  2. % -- Relative units
  3. vh/vw -- Relative units
  4. em -- Relative units (by default 1em == 16px, but it depends on the font size of the element)
  5. rem -- Relative units (by default 1rem == 16px, but it depends on the font size of the root element)
