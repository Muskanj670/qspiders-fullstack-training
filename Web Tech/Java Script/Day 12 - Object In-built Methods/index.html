Object In-built Methods

  1. Object.keys(var) -- Returns an array of keys (property names) from an object.
  2. Object.values(var) -- Returns an array of values from an object.
  3. Object.entries(var) -- Return an array of [key, value] pairs of an object.
  4. Object.fromEntries() -- Converts and array of [key, value] pair back into an object.
  5. Object.freeze(var) -- Freezes an object so that you can not add, delete or change any property. It becomes read only.
  6. Object.isFrozen(var) -- Checks if an object is frozen. Returns true if frozen, else false.
  7. Object.assign() -- Copies all enumerable own properties from one or more source object into a target object, then return a target object.
