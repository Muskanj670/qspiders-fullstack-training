Arrow Function

What is arrow function?
  • An arrow function is just a shorter way to write a function in js.
  • It does not have its own this - it borrows this from wherever it was return.
Why we use Arrow function is JS?
  • For cleaner syntax and content.
  • Avoid this confusion in callback.

  1. Implicit Arrow Function

    2. When you skip return and {} - arrow function return automatically. 
                const sum = (a,b) => a+b;
            const c = sum(10,15);

  2. Explicit Arrow Function

    3. When you use return keyword inside curly braces {}. 
            const sum2 = (x,y) =>{
            const total = x + y;
            console.log(total)
            return total     
        }
        const z = sum2(10,15);
