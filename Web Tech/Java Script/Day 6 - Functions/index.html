Functions in JS

  1. Function is an object in JS
  2. A block of code which is used to perform some specific task.
  3. To call a function we need function reference and parenthesis.
  4. To return something inside from a function we use "return" keyword
  5. If we dose not return anything, function by default returns 'undefined'.
  6. The values passed during function call statement is known as "arguments".
  7. The variable defined in functions definition to store argument is known as "parameters".

Types of functions:

  1. Function Declaration statement (Regular function)
  2. Function as expression / First class function
  3. Immediately invoked function expression (IIFE)
  4. Arrow Function (=>)
    • Implicit return Type
    • Explicit Return Type
  5. Higher order function (HOF)
  6. Callback Function (CF)
