Literals

Literals are different types of values that is used in writing a program.

Types of Literals:

  1. Primitive Literals

    • These are single value literals
    • These are immutable
    • It is of 8 types:

      1. numbers:

        Range: -2^(53) -1 to 2^(53) - 1

      2. bigInt:

        Range: -infinity to +infinity

      3. Nan:

        Stands for Not a Number
        It is computational error given by js engine.
        If we do any operation (like addition, subtraction) with Nan, result will be Nan.

        Example 1:

        const a = 10;
        const b = '5';
        console.log(a - b);
        Output: 5

        Example 2:

        const a = 10;
        const b = '5x';
        console.log(a - b);
        Output: Nan

      4. Boolean:

        Falsy / False Values: A value is falsy if JS treats it as false.
        • false
        • 0
        • 0.0
        • 0n
        • Nan
        • null
        • Undefined
        • ""(Empty String)
        Truthy / True values : A value is truthy if js treats it as true.

      5. null:

        null is an international absence of value (set by you);

      6. undefined:

        Undefined means a variable exists but has not been assigned a value yet(set by js).

        7. Difference between undefined and not defined

        undefined means variable exist but has no value
        not defined means the variable does not exist at all(throws a ReferenceError)

      7. string:

        Strings are collection of characters

        These are of two types:

        • Single Value String : In this we can not have linebreak or white spaces in this.
        • Multi line String / Template String : In this we can have linebreak or white spaces in this. Denoted inside ``

      8. symbol:

        const a = 10;
        const b = symbol(a);
        console.log(b);

  2. Non Primitive Literals

    • These are multi valued
    • These are mutable
    • Types: Function, Array, Object
