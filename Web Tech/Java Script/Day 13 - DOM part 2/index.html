Difference between textContent and innerHTML

textContent

  • Returns only plain text inside an element
  • Ignores html tags - they appear as raw text
  • Faster, safer from XSS attack
  • Does not trigger reflow for html parsing

innerHTML

  • Returns HTML markup inside an element
  • Parses string as html, creating/ removing Dom elements
  • Slower, can be a security risk
  • Trigger reflows after html parsing
