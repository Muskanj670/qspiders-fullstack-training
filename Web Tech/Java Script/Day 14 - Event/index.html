Events

  • Events are the actions performed by the user on browser.

Event Objects

  • Event object is an object created by the browser when user perform some action.
  • Event object holds all the information about which type of event occurred ad the element on which the event occurred
  • Event object is passed to respective event listener on every time event occurred.

Event listener

  • Event listener are the functions that wait for the specific event to occur and then execute JS code(callback) assigned to it.
