Event Propagation

Event propagation is the order in which events travel through the DOM when an event is triggered on an element.

It is happened in three phases:

Capturing

Event travels down from root to target element

Window -> document -> HTML -> Body -> div#GrandParent -> div#Parent -> div#child

Target

Event reaches the actual clicked element

Bubbling

Event travel back up from target to root

Window <- document <- HTML <- Body <- div#GrandParent <- div#Parent <- div#child
