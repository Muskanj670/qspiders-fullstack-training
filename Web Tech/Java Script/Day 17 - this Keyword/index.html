this Keyword

this is a special keyword that refers to the object it belongs to - its value depends on where and how it's used.

  1. this keyword in GEC = window{}
  2. this keyword in regular function = window{}
  3. this keyword in arrow function = window{inherit from GEC}
  4. this keyword in regular method = object{}
  5. this keyword in arrow method = window{}
  6. this keyword in class = object{}
  7. this keyword in constructor function = object{}
  8. this keyword in addEventListner's regular callabck = element{}
  9. this keyword in addEventListner's arrow callback = window
  10. this keyword in nested regular method = window{}
  11. this keyword in nested method, outer is regular and inner is arrow function = object
