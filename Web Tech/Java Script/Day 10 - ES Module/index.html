ES Module

ES Module is the official JavaScript module system introduced in ES6(2015). It lets you split code into separate files and share functionality between them using export and import.

In ES Module we have two types of export:

  1. Default Export

    Used to export a single function, class, or variable from a module. Name can be renamed freely without braces. 

    2.             Example of default export:
            // A.js
            const sum = (a,b) => a+b;
            export default sum;
            // B.js
            import sum from './A.js';
            console.log(sum(2,3)); // Output: 5
  2. Named Export

    Used to export multiple functions, classes, or variables from a module. 

    3.             Example of named export:
            // A.js
            const sum = (a,b) => a+b;
            const product = (a,b) => a*b;
            export { sum, product };
            // B.js
            import { sum, product } from './A.js';
            console.log(sum(2,3)); // Output: 5
            console.log(product(2,3)); // Output: 6
