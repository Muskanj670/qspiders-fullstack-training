var let const

Token

It is smallest unit of programming language.

Example: const a = 5;

const = Keyword

a = Identifier

= = Assignment Operator

5 = Literal

; = Statement Terminator

Keyword

var:

  • Multi time declaration
  • Multi time initialization
let:
  • Single time declaration
  • Multi time initialization
const:
  • Single time declaration
  • Single time initialization
