Normal script (no attribute):

<script src="script.js"></script>

HTML parsing stops -> script downloads + executes -> parsing resumes. Blocking!

Async attribute:

<script async src="script.js"></script>

- Scripts downloads in parallel with html parsing.
- executes immediately when download finishes -> pauses html parsing.
- No guaranteed order if multiple async scripts.

Defer attribute:

<script defer src="script.js"></script>

- scripts downloads in parallel with html parsing.
- executes only after full html is parsed.
- Order is permanent if multiple deferred scripts
