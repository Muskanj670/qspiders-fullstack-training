Type Coercion

To convert one type of data into another type is known as type coercion.

Types of type coercion:

  1. Implicit type coercion

    To convert one type of data into another type implicitly(automatically) by js engine itself.
    Example:

    const a = 10;
    const b = '5';
    console.log(a - b);

    -b will be converted into number automatically by js engine.

  2. Explicit Type coercion:

    To convert one type of data into another type forcefully by js engine
    Here we use some special inbuilt methods:
    • Number()/+
    • Boolean()
    • String()
    • Symbol()
    • BigInt()
