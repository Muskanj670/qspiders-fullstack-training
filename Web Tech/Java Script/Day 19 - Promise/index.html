Promise

Promise is a special type o f object which represents eventual completion of an asynchronous operation.

Promise has 3 states:

  1. pending
  2. fulfilled
  3. rejected

Async /Await

async and await are keywords in js that make working with promises much easier and cleaner. They allow you to write asynchronous code that looks like synchronous code.

  • async - Declares a function as asynchronous. It automatically returns a promise.
  • await - Pauses the execution of the async function until the promise is resolved.
