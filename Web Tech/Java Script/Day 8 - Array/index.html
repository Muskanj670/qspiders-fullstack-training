Array

  1. Array is an object in JS
  2. It is non-primitive type of literal
  3. It is block of memory which is used to store multiple type of values (any type of literal) in same memory block
  4. Array size is dynamic, it means we can store heterogenous type of data and n number of data
  5. Values stored inside array are referred as array elements

Inbuilt methods of Array

  1. Original Value Modified
    • var.push(value) -- Inserts value at the end of the array and return updated length
    • var.pop() -- Deletes the last value of the Array and returns the removed value
    • var.unshift(value) -- It is used to insert element at first index of an array and return updated array length.
    • var.shift() -- It is used to delete element from the first ins=dex and return deleted element
    • var.splice(start, no of elements to be deleted, new values to be added) -- It is used to perform insertion, updation, deletion in array and return the array of deleted elements
  2. Original Value unmodified
    • var.slice(start,stop) -- It is used to copy array element and returns array of copied element.
    • var.indexOf(search value, starting index) -- It is used to get the index of array element, If element is available return element's index else returns -1
