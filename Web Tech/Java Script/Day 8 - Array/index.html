Array

  1. Array is an object in JS
  2. It is non-primitive type of literal
  3. It is block of memory which is used to store multiple type of values (any type of literal) in same memory block
  4. Array size is dynamic, it means we can store heterogenous type of data and n number of data
  5. Values stored inside array are referred as array elements

Inbuilt methods of Array

  1. Original Value Modified
    • var.push(value) -- Inserts value at the end of the array and return updated length
    • var.pop() -- Deletes the last value of the Array and returns the removed value
    • var.unshift(value) -- It is used to insert element at first index of an array and return updated array length.
    • var.shift() -- It is used to delete element from the first ins=dex and return deleted element
    • var.splice(start, no of elements to be deleted, new values to be added) -- It is used to perform insertion, updation, deletion in array and return the array of deleted elements
    • var.reverse() -- It is used to reverse the array
      • Higher Order methods
    • var.sort(callback) -- If callback return -ve value -> it will sort in ascending order
      - If callback return +ve value -> It will sort in descending order
      - If callback return 0 value -> It will not sort
  2. Original Value unmodified
    • var.slice(start,stop) -- It is used to copy array element and returns array of copied element.
    • var.indexOf(search value [,starting index]) -- It is used to get the index of array element, If element is available return element's index else returns -1
    • var.includes(search value [,starting index]) -- It is used to get the boolean output, if element is available return true else false
      • Higher Order methods
    • var.forEach(callback) -- It is used to iterate over array elements and index, It does not return anything so js engine implicitly return undefined
    • var.map(callback) -- It is used to iterate over array, it return a new array and the value returned by callback function will be inserted in new array, if it does not return anything 'undefined' will be stored
    • var.filter(callback) -- It is used to iterate over array, it return a new array. Here, element will be inserted in new array only when callback function returns true.
      • Difference between map and filter? 
                map
            1. Returns a new array of the same length, with each element transformed
            2. Used to modify/transform every element in the array
            Example: [1,2,3].map(n => n * 2) → [2, 4, 6]
         
              filter
          1. Returns a new array of equal or lesser length, keeping only matching elements
          2. Used to remove/select elements based on a condition
          Example: [1,2,3].filter(n => n > 1) → [2, 3]
    • var.reduce(callback) -- Reduces an array down to a single output value (number, string, object, etc.). Uses an accumulator that carries the running result, starting from the initialValue. Iterates through the entire array, applying the callback on each element
    • var.reduceRight(callback) --
    • var.every(callback) -- Test whether all elements in the array pass the condition. Return true only if every element matches, false if any one fails.
    • var.some(callback) -- Return true even a single element pass the condition.
    • var.flat() -- Convert a nested array into a single array by pulling out inner array.
    • Array.isArray(var) -- It is used to check whether array is present inside variable or not. It return boolean value.
    • Array.from(iterable , map_fn) -- Converts any iterable (string, set, map, etc) into an array. It is also used to convert HTML collection and NodeList into an array.
