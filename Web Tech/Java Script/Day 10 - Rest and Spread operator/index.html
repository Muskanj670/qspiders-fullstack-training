Rest and Spread Operator

Rest Operator(...)

The rest operator collects multiple elements and condenses them into a single array. it is used in function parameters or destructuring to gather the 'rest' of the values.

Spread Operator(...)

The spread operator expands an iterable(array, object, string) into individual elements. It is used when passing arguments, building arrays or merging objects.

Why spread operator can perform deep copy of only top level primitive values not nested values? 

      1. Because primitive values (string, number, boolean) are copied by values, not by references, so spread creates independent copies.
      2. But in case of nested objects they are copied by reference using spread operator
