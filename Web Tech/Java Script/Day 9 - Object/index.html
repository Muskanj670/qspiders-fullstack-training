What is Object in JS?

  • An object is a block of memory which has state(variable) and behavior(methods) and where we can store heterogeneous data.
  • An object is a collection of key value pairs that can contain various data types such as number, string, array, function and other objects.
  • In one object we can have multiple key-value pair and it should be separated by comma(,).
  • We can access value of object using dot(.) or square braces ([]).
