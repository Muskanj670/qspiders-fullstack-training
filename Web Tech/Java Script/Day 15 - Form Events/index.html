Form Events

Input Event

Fires every time the va;ue of input changes.

Change Event

Fires only when the input is changed and the element loses focus.

Submit Event

Fires when the form is submitted

Reset Event

Fires when the form is reset (all fields cleaned to default)

Focus Event

Fires when an input gained focus

Blur Event

Fires when an input loses focus

FormData = Definition & Syntax

FormData

FormData is a built-in JavaScript web API that provides a way to construct key-value pairs representing form field and their values. It is primarily used to send data via fetch or XMLHttpRequest, especially for file uploads or mimicking HTML form submission.

Constructor

const formData = new FormData(form?, submitter?);
