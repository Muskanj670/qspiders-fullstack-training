What is JSON?

JSON is a lightweight data format used to store and exchange data. It is human- readable and language-independent, making it the most popular format for sending data between a server and a client.

JSON.stringify()

The JSON.stringify() method converts a JavaScript object or value to a JSON string. Used when you want to send data (e.g., to an API or store in local storage).

JSON.parse

Convert a JSON string to JS object, Used when you receive data (e.g., From an API response or local storage) and want to work with it in your JavaScript code.
